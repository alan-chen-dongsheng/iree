## Onnx的前端

compiler/bindings/python/iree/compiler/tools/import_onnx/
  ├── __main__.py                              # CLI 入口 (9.7KB)
  ├── importer_externalization_overrides.py     # IREE 特定的 ONNX 导入覆盖 (16.4KB)

  核心文件：

  ┌───────────────────────────────────────┬────────────────────────────────────────────────────────────────┐
  │                 文件                  │                              作用                              │
  ├───────────────────────────────────────┼────────────────────────────────────────────────────────────────┤
  │ __main__.py                           │ iree-import-onnx 命令的入口，解析 CLI 参数，调用 ONNX importer │
  ├───────────────────────────────────────┼────────────────────────────────────────────────────────────────┤
  │ importer_externalization_overrides.py │ IREE 自定义的 op 外部化逻辑，处理 ONNX → torch dialect 转换    │
  └───────────────────────────────────────┴────────────────────────────────────────────────────────────────┘

  底层依赖的是 torch-mlir 的 ONNX importer：

  third_party/torch-MLIR/python/torch_mlir/extras/onnx_importer.py

  转换链：ONNX protobuf → onnx_importer.py → IREE overrides → MLIR (torch dialect)

## 使用

```shell
# 编译python的模块
ninja IREECompilerPythonModules
python -m iree.compiler.tools.import_onnx model.onnx -o model.mlir --opset-version 17
```

  ONNX 文件 (.onnx)
        │
        ▼
  ① onnx.load()              ← 用 onnx Python 包读取 protobuf
        │
  ② onnx.shape_inference()   ← 做 shape 推导（填充未标注的维度）
        │
  ③ onnx.version_converter   ← 如果指定了 --opset-version，升级 opset
        │
  ④ onnx_importer.ModelInfo() ← 解析 ONNX 图结构（节点、输入、输出）
        │
  ⑤ model_info.create_module() ← 创建 MLIR Module（空的 MLIR 容器）
        │
  ⑥ NodeImporter.define_function() ← 遍历 ONNX 节点，逐一转为 MLIR op
        │
  ⑦ m.get_asm()              ← 把 MLIR 输出为文本格式 (.mlir)
